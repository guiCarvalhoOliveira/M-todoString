Este projeto feito em java, serviu para meu aprendizado de métodos de String. A ideia do código é fazer uma validação simples de email, em que, se o usuário digitar algo que nao seja em @gmail.com ou @yahoo.com, o codigo nao seguirá em frente.
após validação, o sistema informa o username do usuário, que é o nome antes do "@" e o dominio, que é depois do "@"
